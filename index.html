"Keep improving everyday."

Sovanpisoth Yan (Sotchi) is an aspiring full-stack developer from Phnom Penh, Cambodia, currently majoring in Software Engineering. Passionate about building modern web applications, he focuses on frontend development while continuously learning new technologies and improving his coding skills. This portfolio showcases his projects, experiments, and growth as a developer.

I started programming in 2025, with C as my first language. My first project, completed during my first year of university, was a store management system built entirely in C. Since then, I have become increasingly excited about learning new programming languages and expanding my knowledge in web development, software engineering, and application development. I am passionate about creating systems and applications that can help solve real-world problems.
